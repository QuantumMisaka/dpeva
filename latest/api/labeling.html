Labeling API#

DP-EVA Labeling Module#

This module handles the labeling process for Active Learning, specifically for First Principles (FP) calculations (e.g., ABACUS).

It includes: - Input Generation (Generator) - Resubmission Strategy (Strategy) - Post-processing (PostProcessor)

class dpeva.labeling.AbacusGenerator(config: Dict[str, Any])[源代码]#

基类：object

Generates ABACUS input files for a set of atomic structures.

generate(atoms: Atoms, output_dir: str | Path, task_name: str, stru_type: str | None = None, vacuum_status: List[bool] | None = None, dataset_name: str = 'unknown', system_name: str = 'unknown')[源代码]#

Generate input files for a single structure.

参数:

  • atoms -- ASE Atoms object (preprocessed).

  • output_dir -- Directory to save files.

  • task_name -- Name of the task.

  • stru_type -- Pre-determined structure type (optional).

  • vacuum_status -- Pre-determined vacuum status (optional).

  • dataset_name -- Name of the dataset this task belongs to.

  • system_name -- Name of the system this task belongs to.

class dpeva.labeling.ResubmissionStrategy(attempt_params: List[Dict[str, Any]] | None = None)[源代码]#

基类：object

Manages parameters for each calculation attempt.

This class handles the retrieval and application of parameters for different attempts in the calculation workflow. It allows for modifying input files based on the attempt index.

apply(task_dir: str, attempt: int) → bool[源代码]#

Apply the parameters for the given attempt to the INPUT file.

参数:

  • task_dir -- Directory containing the INPUT file.

  • attempt -- The attempt index (0-based).

返回:

True if applied successfully, False if no params for this attempt.

get_params(attempt: int) → Dict[str, Any][源代码]#

Get parameters for a specific attempt index.
