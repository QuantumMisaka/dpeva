Unified Job Configuration Data Class.

Contains all possible fields for job submission. Follows 'Explicit is better than implicit' principle; core Slurm parameters should be explicitly defined.

class dpeva.submission.JobManager(mode: Literal['local', 'slurm'] = 'local', custom_template_path: str | None = None)[源代码]#

基类：object

Job Submission Manager.

Handles the generation of submission scripts and execution of jobs for both local and Slurm environments.

generate_script(config: JobConfig, output_path: str) → str[源代码]#

Generate a job script from configuration and write it to a file.

参数:

  • config (JobConfig) -- The job configuration object.

  • output_path (str) -- The path to save the generated script.

返回:

The absolute path to the generated script.

返回类型:

str

submit(script_path: str, working_dir: str = '.') → str[源代码]#

Submit the job script.

参数:

  • script_path (str) -- Path to the submission script.

  • working_dir (str, optional) -- Directory to execute the submission from. Defaults to ".".

返回:

Output from the submission command (e.g. job ID).

返回类型:

str

抛出:

subprocess.CalledProcessError -- If submission fails.

submit_python_script(script_content: str, script_name: str, job_config: JobConfig, working_dir: str = '.') → str[源代码]#

Helper to write a python script and submit it as a job.

参数:

  • script_content (str) -- The Python code to run.

  • script_name (str) -- Name of the python file (e.g. "run_task.py").

  • job_config (JobConfig) -- Configuration for the job. The 'command' field will be overridden to run the python script.

  • working_dir (str) -- Directory to write script and submit job.

返回:

Submission result.

返回类型:

str
