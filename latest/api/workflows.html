class dpeva.workflows.FeatureWorkflow(config: Dict | FeatureConfig)[源代码]#

基类：object

Workflow for generating atomic/structural descriptors using DeepPot. Refactored using DDD Managers.

run()[源代码]#

Executes the feature generation workflow.

  1. Configures workflow logging.

  2. Validates data paths.

  3. Generates descriptors via CLI (dp eval-desc) or Python API (DeepPot).

  4. Handles local or Slurm execution.

class dpeva.workflows.InferenceWorkflow(config: Dict | InferenceConfig, config_path: str | None = None)[源代码]#

基类：object

Workflow for running inference using an ensemble of DPA models. Supports both Local and Slurm backends via JobManager. Refactored using DDD Managers.

analyze_results()[源代码]#

Analyze results for all models.

run()[源代码]#

Executes the inference workflow.

  1. Configures workflow logging.

  2. Discovers trained models.

  3. Submits parallel inference jobs (dp test) via Slurm or Local backend.

  4. If Local, automatically triggers analysis.

class dpeva.workflows.TrainingWorkflow(config: Dict | TrainingConfig)[源代码]#

基类：object

Workflow for parallel fine-tuning of DeepMD models. Supports both local multiprocessing and Slurm submission. Refactored using DDD Managers.

run()[源代码]#

Executes the training workflow.

  1. Configures workflow logging.

  2. Generates task-specific configurations and seeds.

  3. Creates task directories and copies base models.

  4. Generates submission scripts (Slurm or Local).

  5. Submits jobs to the execution backend.
