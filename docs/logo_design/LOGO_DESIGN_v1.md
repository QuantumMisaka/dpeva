# DP-EVA Logo 设计需求文档

**版本**: v1.0  
**日期**: 2026-03-01  
**项目**: DP-EVA (Deep Potential EVolution Accelerator)

---

## 1. 项目背景与愿景

### 1.1 项目简介
- **全称**: Deep Potential EVolution Accelerator
- **简称**: DP-EVA
- **中文名**: 深度势能演化加速器
- **核心功能**: 基于主动学习（Active Learning）的高效微调框架，专为 DPA3 (Deep Potential v3) 模型设计。
- **核心目标**: 从覆盖大规模化学空间的海量结构样本中，像“淘金”一样精准识别出对模型能力提升最关键的高价值样本。

### 1.2 核心理念
- **精准 (Precision)**: 在海量数据中精确定位高价值样本。
- **演化 (Evolution)**: 通过不断的迭代采样，模型能力像生物进化一样不断增强。
- **独立与认同**: 取名 "EVA" 的灵感来源于《新世纪福音战士》（Neon Genesis Evangelion）。在番剧中，个体需要在 LCL 之海（象征混沌、无差别的原始汤）中维持自我边界（AT Field）以保持独立存在。对应到本项目，即在**茫茫化学空间（混沌之海）**中，通过算法识别出**最具代表性的样本（独立个体）**。

---

## 2. 设计风格与总体编排

### 2.1 整体结构
Logo 的整体编排应参考项目内置的 ASCII Banner 结构：
- **左侧**: `DP` (代表 Deep Potential / DeepModeling)
- **右侧**: `EVA` (代表 Evolution Accelerator / Evangelion)
- **连接**: 两者之间应有视觉上的连接或过渡，体现“基于 DP 的演化”。

参考结构（ASCII Banner）：
```
  DP  ____   EVA
```

### 2.2 关键词
`硬核科技 (Hard Sci-Tech)` | `生物机械 (Bio-mechanical)` | `进化 (Evolution)` | `精准 (Precision)`

## 3. 详细设计元素

### 3.1 "DP" 部分 - 传承 DeepModeling 基因
- **设计参考**: 见附件 `deepmodeling.png`。
- **风格**: 几何、严谨、科学、结构化。
- **视觉元素**:
  - **六边形 (Hexagon)**: 象征苯环、晶体结构、科学计算。
  - **颜色**: DeepModeling 品牌蓝（科技蓝、深蓝）。
  - **字体**: 无衬线体，粗壮有力，体现工业级软件的稳重。
- **要求**: "DP" 两字的设计应与 DeepModeling 社区的视觉语言保持一致，体现项目作为该生态一部分的身份。

### 3.2 "EVA" 部分 - 致敬《新世纪福音战士》
- **设计灵感**: 《新世纪福音战士》（Neon Genesis Evangelion）。
- **风格**: 锐利、动态、生物感、觉醒。
- **视觉元素**:
  - **机体特征**: 初号机（Unit-01）的头部轮廓、独角、下颚。
  - **AT Field**: 八边形光盾效果，象征“筛选”与“界限”。
  - **颜色**: 初号机配色（紫 #6B3E99、绿 #39D945）或 警报红/橙（象征高能/LCL）。
- **具体要求**:
  - **字母 "A"**: 请将字母 "A" 设计为 **EVA 初号机头部的抽象化剪影**（侧面或正面均可，需体现独角特征）。
  - **字母 "E" & "V"**: 字体风格应比 "DP" 更具动感和锐度，体现“加速 (Accelerator)”的速度感。

### 3.3 融合与过渡
- **概念融合**: 从 "DP" 的**六边形（化学分子）**过渡到 "EVA" 的**八边形（AT Field / 筛选边界）**。
- **视觉叙事**: 左侧是冷静的科学计算（蓝），右侧是进化的爆发力（紫/绿/橙）。

## 4. 配色方案 (Color Palette)

建议采用双色调或三色调渐变：

1.  **DeepModeling Blue (主色调 1)**
    - 用于 "DP" 部分。
    - 象征：深度势能、基础科学、严谨。
    - 参考色值: `#1E3D59` (深蓝) 或 DeepModeling Logo 原色。

2.  **EVA Unit-01 Purple (主色调 2)**
    - 用于 "EVA" 部分主体。
    - 象征：神秘、进化、核心力量。
    - 参考色值: `#7058A3` (EVA紫)。

3.  **Fluorescent Green (点缀色)**
    - 用于 "EVA" 的细节（如 "A" 的眼睛或线条）。
    - 象征：生物能、激活、高亮。
    - 参考色值: `#39D945` (荧光绿)。

4.  **LCL Orange (辅助色/背景)**
    - 用于背景光晕或连接线条。
    - 象征：化学空间、原始汤、生命之海。
    - 参考色值: `#FFA500` (琥珀色)。
