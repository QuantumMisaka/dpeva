## 回归脚本与CI接入

<a id="ci-regression"></a>

- 脚本路径：`scripts/collection_plot_regression.py`
- 作用：校验图名是否属于标准集合、尺寸是否匹配fig_dpi、DPI/色彩模式是否一致。
- CI示例命令：
```bash
python scripts/collection_plot_regression.py \
  --view-dir test/dpeva-iter1/dpeva_post_joint/view \
  --view-dir test/dpeva-iter1/dpeva_post_normal/view \
  --fig-dpi 300 \
  --strict-native \
  --json-out test/collection_plot_regression.json
```
- 历史样本目录`test/stage8-2-used/dpeva_uqft4w/view`缺少现行Collection配置文件，本报告按图像内嵌DPI=150进行比对。
