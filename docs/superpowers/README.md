# Superpowers Execution Documents

This directory stores agentic execution plans and design drafts that still need
to remain close to their original Superpowers task format.

Stable, team-facing plans should be promoted to `docs/plans/`. Release-time
historical records should be moved to `docs/archive/vX.Y.Z/`.
