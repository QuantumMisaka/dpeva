# Governance（文档治理）

- Status: active
- Audience: Maintainers / Developers
- Last-Updated: 2026-02-19

本目录用于存放文档体系的治理交付物：规划、审计、追踪矩阵、库存清单与相关工具配置。

## 1. 子目录速览

| 目录 | 用途 |
|---|---|
| `/docs/governance/plans/` | 路线图与可执行规划（带日期版本）。 |
| `/docs/governance/audits/` | 审计/评审报告（带日期版本）。 |
| `/docs/governance/traceability/` | 功能-文档等双向追踪矩阵（活文档）。 |
| `/docs/governance/inventory/` | 文档库存/清单（建议自动生成或定期复核）。 |
| `/docs/governance/tools/` | 文档检查与链接规范化相关配置（机器可读）。 |
