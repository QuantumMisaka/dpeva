# Inventory（文档库存与责任清单）

本目录维护文档资产清单与 Owner 覆盖矩阵，用于稳态治理阶段的责任追踪与复核。

## 1. 核心文件

- 文档目录清单：`docs-catalog.md`
- Owner 责任矩阵：`owners-matrix.md`

## 2. 使用方式

- 新增 `status: active` 文档后，同步检查是否存在 `owner` 或 `owners`
- 按目录域对照 `owners-matrix.md` 校验责任归属
- 每月复核一次库存清单与责任覆盖
