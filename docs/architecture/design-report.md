本文件作为 `docs/architecture/` 下的“架构分析报告”落点占位，用于后续把历史设计分析逐步拆解为：

当前权威内容仍位于：

- [docs/design/DP-EVA_Design_Report.md](/docs/design/DP-EVA_Design_Report.md)

- 现阶段请直接阅读 design 报告；后续逐步迁移到 architecture 体系。

## 4. 异常处理
